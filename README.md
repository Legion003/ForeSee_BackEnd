# ForeSee后端

## 项目介绍

### 组织结构

```lua
ForeSee
├── client-121 -- 用于接收从前端发过来的请求
├── commons -- 工具类及通用代码
├── server-192 -- 数据检索
```

### 技术选型

| 技术       | 说明         | 官网                                   |
| ---------- | ------------ | -------------------------------------- |
| SpringBoot | 容器+MVC框架 | https://spring.io/projects/spring-boot |
| Redis      | 缓存         | https://redis.io/                      |
| MongoDb    | NoSql数据库  | https://www.mongodb.com                |
| Lombok     | 封装工具     | https://github.com/rzwitserloot/lombok |
